# rust-zaim
rust-zaim is a Rust client library for the [Zaim API](https://dev.zaim.net/home)

## Usage

### Installation

#### Set Enviroments
1. Copy `.env`.

```zsh
$ cp .env.example .env
$ cat .env
ZAIM_BASE_URL=
ZAIM_CUSTOMER_ID=
ZAIM_CUSTOMER_SECRET=
ZAIM_TOKEN=
ZAIM_SECRET=
ZAIM_VERIFIER=
```

2. Obtain credentials on the [Zaim Developers Center](https://dev.zaim.net/) and set values.
